Calculator Guide

Simple Mode - Basic Mathematics

  • Numbers: 0-9, 00, 000 for quick entry, decimal point (.)
  • Basic Operations: +, −, ×, ÷
  • Powers & Roots: x² (square), √ (square root)
  • Parentheses: ( ) for grouping operations
  • Functions: 1/x (reciprocal), ± (change sign), % (percentage)
  • Clear (C): Reset calculator
  • Delete (←): Remove last digit
  • Equals (=): Calculate result (or press Enter)

Toggle to Advanced Mode for trigonometry, logarithms, and more!

Advanced Mode - All Functions

Trigonometry
  • sin, cos, tan: Trigonometric functions
  • sin⁻¹, cos⁻¹: Inverse trig functions
  • Toggle Degrees/Radians above
Logarithms & Exponentials
  • log: Base 10 logarithm
  • ln: Natural logarithm
  • e^x: e to the power x
  • 10^x: 10 to the power x
Powers & Roots
  • √: Square root
  • ∛: Cube root
  • x², x³: Square, cube
  • x^y: x to the power of y
Advanced Functions
  • |x|: Absolute value
  • n!: Factorial
  • 1/x: Reciprocal
  • mod: Modulo operation
  • ⌊x⌋, ⌈x⌉: Floor, ceiling
Constants
  • π (pi): 3.14159...
  • e: 2.71828... (Euler's number)
