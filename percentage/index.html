Percentage Formulas

  • What is X% of Y? Result = (X / 100) × Y
  • X is what % of Y? Percentage = (X / Y) × 100
  • X is Y% of what? Whole = (X × 100) / Y
  • Percentage Increase: New Value = Original + (Original × Increase% / 100)
  • Percentage Decrease: New Value = Original - (Original × Decrease% / 100)
  • Percentage Change: Change% = ((New - Old) / Old) × 100
